## Features / 기능

| Feature | Description | 설명 |
|---------|-------------|------|
| **Clipboard History** | Stores up to 1,000 copied items. Press `Cmd+Shift+V` to open at cursor position | 최대 1,000개 클립보드 저장. `Cmd+Shift+V`로 커서 위치에서 바로 꺼냄 |
| **App Switcher** | Switch between running apps instantly with `Option+Tab` | `Option+Tab`으로 실행 중인 앱을 빠르게 전환 |
| **Screenshot** | Capture full screen, region, or window — one shortcut each | 전체 화면 / 영역 / 창 캡처 — 단축키 하나로 |
| **Window Snap** | Snap windows to any position with a single hotkey | 창을 원하는 위치에 단축키 하나로 고정 |

All shortcuts are fully customizable in Settings.
모든 단축키는 설정에서 자유롭게 변경할 수 있습니다.

## Installation / 설치

### Homebrew (Recommended / 권장)

```bash
brew tap supaicy/bicmac-tap
brew install --cask bicmac
```

### Update / 업데이트

```bash
brew update && brew upgrade --cask bicmac
```

### Uninstall / 삭제

```bash
brew uninstall --cask bicmac
```

## Security / 보안

BicMac is open source. You can read every line of code before installing.
BicMac은 오픈소스입니다. 설치 전에 모든 코드를 직접 확인할 수 있습니다.

- Source code / 소스코드: [github.com/supaicy/BicMac](https://github.com/supaicy/BicMac)
- Binaries / 바이너리: [github.com/supaicy/BicMac-releases](https://github.com/supaicy/BicMac-releases)
- Built locally on a Mac — no third-party build servers involved
- 빌드는 로컬 Mac에서 직접 수행하며 제3자 서버를 거치지 않습니다

> **Gatekeeper notice:** BicMac is distributed with ad-hoc signing. On first launch, macOS may show a security warning. Go to **System Settings → Privacy & Security → Open Anyway** to allow it.
>
> **Gatekeeper 안내:** ad-hoc 서명으로 배포됩니다. 첫 실행 시 보안 경고가 뜰 수 있습니다. **시스템 설정 → 개인 정보 보호 및 보안 → 확인 없이 열기** 로 허용할 수 있습니다.
