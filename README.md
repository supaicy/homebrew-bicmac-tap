## 기능

| 기능 | 설명 |
|------|------|
| **클립보드 히스토리** | 복사한 내용을 최대 1,000개까지 저장. `Cmd+Shift+V`로 커서 위치에서 바로 꺼냄 |
| **앱 전환기** | `Option+Tab`으로 실행 중인 앱을 빠르게 전환 |
| **스크린샷** | 전체 화면 / 영역 / 창 캡처 — 단축키 하나로 |
| **윈도우 스냅** | 창을 원하는 위치에 빠르게 고정 |

모든 기능은 설정에서 단축키를 자유롭게 변경할 수 있습니다.

## 설치

### Homebrew (권장)

```bash
brew tap supaicy/bicmac-tap
brew install --cask bicmac
```

### 업데이트

```bash
brew update && brew upgrade --cask bicmac
```

### 삭제

```bash
brew uninstall --cask bicmac
```

## 시스템 요구 사항
